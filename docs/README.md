## 📋 技术栈

### 后端技术
- **语言**: Go 1.21+
- **框架**: Gin + GORM
- **数据库**: MySQL (主) / SQLite (备)
- **日志**: Logrus

### 前端技术
- **框架**: Vue.js 3.0 + TypeScript
- **UI组件**: Ant Design Vue 3.0
- **状态管理**: Pinia
- **路由**: Vue Router 4.0
- **HTTP客户端**: Axios
- **构建工具**: Vite
